## generalui介绍
#### generalui是一款基于react的功能强大的业务组件库，目前部分组件基于antdesign进行的二次开发，支持按需导入，可定制。组件划分为:
* 页面组件(PageComponent)
* 功能组件(FuncComponent)
* 引擎组件(EngineComponent)
* 公共Icon
* 自研通用请求库(request.js)

* react-dom: 6.8.6
